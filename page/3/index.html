谈谈css的overflow属性

发表于 2015-09-25   |   分类于 前端   |  

overflow属性取值

  1. visible：内容不会被剪裁，内容超出容器时，可以被渲染到外面
  2. hidden：内容会被剪裁，滚动条也不能展示出被裁减的内容
  3. scroll：内容将被剪裁，但可以使用滚动条来查看被裁掉的内容
  4. auto：内容是否被裁减由浏览器具体实现决定
